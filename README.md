# Chatbot
This is a simple C# console program that prints a welcome message to the screen.
The line using System; allows the program to use basic features like printing text with Console.WriteLine.
The public class start defines a class, which is required in C# to hold the program code.
The Main method is where the program starts running. Every C# program needs a Main method as its entry point.
Inside Main, the Console.WriteLine lines display text in the console. They create a formatted welcome message for a chatbot.
Overall, the program does not actually chat—it just shows a greeting message when you run it.
